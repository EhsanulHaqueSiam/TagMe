4. Create `functions/src/onMessageCreated.ts`:
   - Import `onDocumentCreated` from `firebase-functions/v2/firestore`
   - Import `getMessaging` from `firebase-admin/messaging`
   - Import `getFirestore` from `firebase-admin/firestore`
   - Export `onMessageCreated` function triggered on `"conversations/{conversationId}/messages/{messageId}"`
   - Implementation:
     a. Get snapshot data, return if null
     b. Extract senderId, senderName, text, type from message data
     c. Get conversation document to find participantIds
     d. Find recipientId: participantIds.find(id => id !== senderId). Return if no recipient.
     e. Get recipient's FCM tokens from `students/{recipientId}/tokens`
     f. Map tokens to string array. Return if empty.
     g. Send FCM via `getMessaging().sendEachForMulticast({tokens, notification: {title: senderName, body: type === 'phone_shared' ? 'Shared their phone number' : text}, data: {type: 'chat_message', conversationId: event.params.conversationId}, android: {priority: 'high'}})`
     h. Handle invalid tokens: iterate results, if result.success is false and error code is 'messaging/registration-token-not-registered' or 'messaging/invalid-registration-token', delete the stale token document from Firestore
     i. Increment unread count for recipient: `update({['unreadCounts.${recipientId}']: FieldValue.increment(1)})` on the conversation document

5. Create `functions/src/onRequestAccepted.ts`:
   - Import `onDocumentUpdated` from `firebase-functions/v2/firestore`
   - Import `getMessaging` from `firebase-admin/messaging`
   - Import `getFirestore` from `firebase-admin/firestore`
   - Export `onRequestAccepted` function triggered on `"joinRequests/{requestId}"`
   - Implementation:
     a. Get before and after data. Return if either is null.
     b. Check if status changed TO 'accepted': `before.data()?.status !== 'accepted' && after.data()?.status === 'accepted'`. Return if not a status change to accepted.
     c. Extract requesterId, requesterName from after data
     d. Get the ride document from `rides/{rideId}` to get ride details (posterName, originAddress, destinationAddress, departureTime)
     e. Get requester's FCM tokens from `students/{requesterId}/tokens`
     f. Send FCM: `{notification: {title: '${posterName} accepted your ride request', body: '${originAddress} -> ${destinationAddress}, ${formattedTime}'}, data: {type: 'ride_match', rideId: rideId}, android: {priority: 'high'}}`
     g. Handle stale tokens same as onMessageCreated
     h. Also send notification to the poster about the match: get poster's tokens from `students/{posterId}/tokens`, send `{title: '${requesterName} joined your ride', body: '${originAddress} -> ${destinationAddress}'}`

6. Create `functions/src/index.ts`:

   - Re-export both functions:
     ```typescript
     export { onMessageCreated } from "./onMessageCreated";
     export { onRequestAccepted } from "./onRequestAccepted";
     ```
