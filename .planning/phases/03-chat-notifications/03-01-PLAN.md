2. Create `lib/features/chat/data/models/message.dart`:
   - Freezed model with fields: `String senderId` (required), `String senderName` (required), `String text` (required), `@Default('text') String type` (values: 'text', 'phone_shared', 'system'), `String? id`, `String? phoneNumber` (only for phone_shared type), `DateTime? createdAt`
   - Same part directives pattern

3. Create `lib/features/chat/data/repositories/chat_repository.dart`:
   - Provider: `@riverpod ChatRepository chatRepository(Ref ref)` returning `ChatRepository(firestore: FirebaseFirestore.instance)`
   - Class `ChatRepository` with `FirebaseFirestore _firestore` field
   - Private getter: `_conversationsCollection => _firestore.collection('conversations')`
   - Method `Future<String> createConversation({required String rideId, required String posterId, required String requesterId, required String rideOrigin, required String rideDestination, required String rideTransportType, required DateTime rideDepartureTime})`:
     - Check for existing conversation with same rideId + both participants using `.where('rideId', isEqualTo: rideId).where('participantIds', arrayContains: posterId).get()` then iterate results to check if requesterId is in participantIds list
     - If exists, return existing doc.id
     - Otherwise create new doc with: participantIds array, rideId, rideOrigin, rideDestination, rideTransportType, rideDepartureTime as Timestamp, lastMessage: 'Ride match! Start chatting.', lastMessageAt: FieldValue.serverTimestamp(), lastSenderId: '', unreadCounts map (both IDs -> 0), createdAt: FieldValue.serverTimestamp()
     - Add system message to messages subcollection: senderId: '', senderName: 'System', text: 'You matched on this ride. Say hi!', type: 'system', createdAt: FieldValue.serverTimestamp()
     - Return docRef.id
   - Method `Stream<List<Conversation>> conversationsForStudent(String studentId)`:
     - Query conversations where participantIds arrayContains studentId, ordered by lastMessageAt descending
     - Map snapshots to List<Conversation> using _conversationFromDoc helper
   - Method `Stream<List<Message>> messagesForConversation(String conversationId, {int limit = 30})`:
     - Query messages subcollection ordered by createdAt descending, limited
     - Map snapshots to List<Message> using _messageFromDoc helper
   - Method `Future<void> sendMessage({required String conversationId, required String senderId, required String senderName, required String text, String type = 'text', String? phoneNumber})`:
     - Use batch: add message doc to subcollection, update conversation metadata (lastMessage, lastMessageAt, lastSenderId)
     - For phone_shared type, set lastMessage to 'Shared phone number'
     - Include phoneNumber field in message doc only if non-null
   - Method `Future<void> markAsRead(String conversationId, String studentId)`:
     - Update `unreadCounts.$studentId` to 0
   - Method `Future<Conversation?> getConversation(String conversationId)`:
     - Fetch single doc, return null if not exists
   - Helper `_conversationFromDoc`: extracts doc data, sets id from doc.id, converts Timestamp fields (lastMessageAt, rideDepartureTime, createdAt) to ISO8601 string for Freezed
   - Helper `_messageFromDoc`: extracts doc data, sets id from doc.id, converts createdAt Timestamp to ISO8601 string

4. Create `lib/features/chat/providers/chat_providers.dart`:
   - `@riverpod Stream<List<Conversation>> conversationList(Ref ref, String studentId)`: watches chatRepositoryProvider, returns conversationsForStudent(studentId)
   - `@riverpod Stream<List<Message>> chatMessages(Ref ref, String conversationId, {int limit = 30})`: watches chatRepositoryProvider, returns messagesForConversation(conversationId, limit: limit)
   - `@riverpod int totalUnreadCount(Ref ref, String studentId)`: watches conversationListProvider(studentId), sums all unreadCounts[studentId] values from conversations. Return 0 if loading/error.
   - `@riverpod Future<Conversation?> conversationDetail(Ref ref, String conversationId)`: watches chatRepositoryProvider, returns getConversation(conversationId)

5. Run `dart run build_runner build --delete-conflicting-outputs` to generate .freezed.dart and .g.dart files.
