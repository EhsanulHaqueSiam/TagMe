    1. Update `android/app/build.gradle.kts` to add release signing config:
       - Add these imports at the TOP of the file (before plugins block):
         ```kotlin
         import java.util.Properties
         import java.io.FileInputStream
         ```
       - Add keystore properties loading AFTER the plugins block, BEFORE android block:
         ```kotlin
         val keystoreProperties = Properties()
         val keystorePropertiesFile = rootProject.file("key.properties")
         if (keystorePropertiesFile.exists()) {
             keystoreProperties.load(FileInputStream(keystorePropertiesFile))
         }
         ```
       - Inside the `android { }` block, add signingConfigs BEFORE buildTypes:
         ```kotlin
         signingConfigs {
             create("release") {
                 keyAlias = keystoreProperties["keyAlias"] as String? ?: ""
                 keyPassword = keystoreProperties["keyPassword"] as String? ?: ""
                 storeFile = keystoreProperties["storeFile"]?.let { file(it as String) }
                 storePassword = keystoreProperties["storePassword"] as String? ?: ""
             }
         }
         ```
         Note: Using `as String? ?: ""` fallback so the build does not crash when key.properties is absent (dev machines without keystore).
       - Update the release buildType:
         ```kotlin
         buildTypes {
             release {
                 signingConfig = if (keystorePropertiesFile.exists()) {
                     signingConfigs.getByName("release")
                 } else {
                     signingConfigs.getByName("debug")
                 }
             }
         }
         ```
         This falls back to debug signing when key.properties is missing (development), but uses release signing for actual builds.

    2. Create `android/key.properties.example` with template and instructions:
       ```properties
       # Release signing configuration for TagMe
       # Copy this file to key.properties and fill in your values.

       #
       # Generate your upload keystore:
       #   keytool -genkey -v -keystore ~/tagme-upload-keystore.jks \
       #     -keyalg RSA -keysize 2048 -validity 10000 -alias upload
       #
       # Back up your keystore file and passwords securely!
       # If you lose the upload key, contact Google Play support to reset it.

       storePassword=YOUR_STORE_PASSWORD
       keyPassword=YOUR_KEY_PASSWORD
       keyAlias=upload
       storeFile=/path/to/your/tagme-upload-keystore.jks
       ```
