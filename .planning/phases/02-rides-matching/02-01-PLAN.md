     - Calls `_client.directionsRouteCoordsGet` with start/end ORSCoordinate
     - Sums Haversine distances between consecutive polyline points for road distance
     - Returns `RouteData` record/class with `distanceKm` (double) and `polylinePoints` (List<LatLng>)

     - Calls ORS Pelias geocoding via the client
     - Returns address string or 'Unknown location' on failure

   - Add `@riverpod` provider: `RouteService routeService(Ref ref)` returning `RouteService(apiKey: orsApiKey)` where `orsApiKey` is a constant (placeholder empty string -- user must configure)
