# Icon Assets

Place these files here before running icon/splash generation:

- icon_foreground.png (1024x1024, transparent background, white icon)
  Used for: Adaptive launcher icon foreground and monochrome icon
  Run: dart run flutter_launcher_icons

- splash_logo.png (1152x1152, transparent background)
  Used for: Native splash screen
  Run: dart run flutter_native_splash:create

The adaptive icon will use #1B73E8 (TagMe accent blue) as the background color.
